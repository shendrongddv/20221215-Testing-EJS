# Ejs-Boilerplate
Turning html template (https://html5up.net/phantom) into a Nodejs Ejs template boilerplate.

# Directory structure
```bash
Html directory
├───phantom
│   ├───assets
│   │   ├───css
│   │   ├───fonts
│   │   ├───js
│   │   └───sass
│   │       ├───base
│   │       ├───components
│   │       ├───layout
│   │       └───libs
│   └───images
```
Project directory
``` bash
├───node_modules
├───public
│   ├───assets
│   │   ├───css
│   │   ├───fonts
│   │   ├───js
│   │   └───sass
│   │       ├───base
│   │       ├───components
│   │       ├───layout
│   │       └───libs
│   └───images
├───routes
└───views
    ├───pages
    └───partials
```

# Installation
```
git clone https://github.com/apeironla/Ejs-Boilerplate.git
npm install
node server 
```
